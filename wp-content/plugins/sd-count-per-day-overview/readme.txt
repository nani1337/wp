== Description ==

Provides an overview of visitor statistics for all blogs, using Count Per Day. Requires that <a href="http://wordpress.org/extend/plugins/count-per-day/">Thomas Braider's Count Per Day plugin</a> is installed.

The data table is refreshed automatically after the table data becomes too old and can also be manually refreshed by the network admin.

Available in:

* English
* Swedish


== Installation ==

1. Unzip and copy the zip contents (including directory) into the `/wp-content/plugins/` directory
1. Activate the plugin sitewide through the 'Plugins' menu in WordPress.
